HTML

  • HTML is a markup language. Elements are wrapped between tags.
  • The head element contains metadata about the webpage.
  • The body element represents the visible content shown to the user
  • The headers are ordered like "h1" or "h2", etc.
  • Tags are contained within angled brackets, and a forward slash is used to close an element tag: "<>" and "/".
  • Most body text is wrapped with "<p>text here</p>
  • Nesting elements within one another is possible, such as using bold text within this unordered list.
  • Inserting a hyperlink is done like this: <a href="link">"display text"</a>.
